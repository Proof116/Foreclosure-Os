Airtable is the central database for all your deals. Everything flows through it — ingest, scoring, alerts, pipeline, and profit tracking.
⏱ ~30 minutes
1
Create your Airtable account & base
Start here
1
Sign up / log in to Airtable
Go to airtable.com → sign up free (Plus plan at $20/mo recommended for automation volume)
2
Create a new base
Click "+ Add a base" → "Start from scratch" → name it "Foreclosure OS"
3
Get your Base ID
Look at the URL: airtable.com/appXXXXXXXXXXXXX/... — copy the part starting with "app". That's your Base ID.
↗ Open Airtable
2
Create the 3 required tables
Important
⚠Table names must match exactly — the dashboard and Make.com look for these exact names.
1
Table 1: "Raw Leads"
Rename the default table to Raw Leads. Add these fields:
Property Address → Single line text
City → Single line text
State → Single line text
County → Single line text
Opening Bid ($) → Currency
AVM Estimate ($) → Currency
Sale Date → Date
Foreclosure Stage → Single select: NOD, NTS, REO
Source → Single line text
Created → Created time (auto)
2
Table 2: "Qualified Leads"
Add a new table named exactly Qualified Leads. Add all Raw Lead fields, plus:
Score → Number (0–100)
MAO → Currency
Equity % → Percent
Days to Sale → Number
Alert Sent? → Checkbox
Phone → Phone number
Lead Status → Single select: HOT, WARM, COLD, NEW
Replied → Checkbox
3
Table 3: "Active Pipeline"
Add a new table named Active Pipeline. Add all Qualified Lead fields, plus:
Pipeline Stage → Single select: Monitoring, Notice of Sale, Auction Pending, Under Review, Won, Dead
Strategy → Single select: Flip, Rental, Wholesale, BRRRR
ARV ($) → Currency
Rehab ($) → Currency
Est Profit → Currency
ROI % → Percent
Assigned To → Single line text
Notes → Long text
3
Generate your API token
Quick
1
Go to your Airtable account settings
airtable.com/account → scroll to API section → click "Generate API token"
2
Name it and set permissions
Name: foreclosure-os → Scope: data.records:read and data.records:write → Access: your Foreclosure OS base only
3
Copy the token (starts with "pat")
You'll use this in both the dashboard and Make.com. Save it in your .env file and password manager.
↗ Open Airtable Account

★ Phase 6 — Tools
Use All 5 New Features
Here's exactly how to use each new tool added in v2 — start to finish.
⏱ Read time: 10 min
◆
MAO Calculator — How to use it
1
Pick your strategy first
Click one of the 4 tabs at top: Fix & Flip, Buy & Hold, Wholesale, or BRRRR. The form adapts to show only the relevant fields.
2
Enter the 3 key numbers
ARV (what it's worth fixed up) · Opening Bid (what you'd pay at auction) · AVM (current estimated value). MAO calculates instantly.
3
Adjust sliders to model scenarios
Drag the Rehab, Margin, Closing, and Holding sliders. The right panel updates live with your MAO, bid comparison, and profit estimate.
4
Check the "Scenario Comparison" at bottom right
It shows MAO for all 4 strategies side by side — instantly see if the deal is better as a flip vs rental.
5
Send to Ranking with one click
Click "→ Send to Ranking" — the deal appears in your Deal Ranking table, auto-scored, ready to compare against others.
★
Deal Ranking — How to use it
1
Deals auto-populate from demo data
8 demo deals are preloaded so you can see the system immediately. When Airtable is connected, your live deals will replace these.
2
Filter by intent
Click 🔴 HOT, 🟡 WARM, or 🔵 WATCH chips to filter. HOT = score ≥ 85. Those are your priority deals.
3
Sort by what matters today
Use the sort dropdown: Days to Sale when you need to act fast. Est. Profit to focus on biggest upside. ROI for capital efficiency.
4
Use the action buttons on each row
MAO → instantly loads deal into the calculator. Nego → loads deal into the AI Negotiation assistant with property context pre-filled.
5
Add custom deals with + Add Deal
Click the green button → fill in the form → system auto-scores it and drops it into the ranked list.
◎
AI Negotiation Assistant — How to use it
1
Load a property first
Click "Load Demo" to see a preloaded property, or hit "Nego" from Deal Ranking to load a real deal. Property stats populate the left panel.
2
Paste the seller's incoming SMS
When a seller texts back, paste their exact message into the chat box and press Send (or Enter). The AI coach instantly classifies their intent and gives you a reply script.
3
Read the Intent badge
Top right of the chat shows 🔴 HOT / 🟡 WARM / ⚪ COLD. HOT = call them immediately. Warm = follow up in 48 hours. Cold = pause outreach.
4
Use the pre-built scripts on the left
The left sidebar has 3 scripts: Opener (first contact) · Price Anchor (includes your MAO) · Cold Re-engagement. Click copy and send via Twilio or your phone.
5
Use the "Suggest" chips for coaching
Bottom bar has Opener · Follow-up · Close chips. Click them to get a coaching message from the AI with specific language for that stage.
$
Profit Dashboard — What to look at
1
Switch periods at top right
Toggle between MTD / QTD / YTD / All Time. All 8 metric cards update instantly.
2
Check ROI by Strategy chart
This tells you which strategy is performing best for you. Wholesale often wins on ROI %. Flip wins on absolute dollars. Adjust your buy-box based on this.
3
Use "Top Deals by Profit" as your priority list
The 5 highest-profit deals are listed. Cross-reference with the Ranking system to make sure you're spending time on the right ones.
4
Pipeline Value Breakdown
The bar chart shows dollar value locked in each stage. If too much is sitting in "Under Review" — push those deals to a decision. Time kills deals.
⬡
Intelligence System — Daily routine
1
Check the Heat Map first thing each morning
Red states = highest activity and most opportunity. Hover any state cell to see starts, REOs, and score. Focus your outreach on red/orange states.
2
Read the Intelligence Feed
Six market signals updated regularly. URGENT tags = act now. OPPORTUNITY = competitor pulling back. COMPETITOR = adjust your bids.
3
Watch the Competitor Activity bars
If Invitation Homes or Progress Residential activity rises, you'll be outbid. Shift to smaller markets (under $300k) or different counties where they're not active.
4
Read the AI Market Summary
Generated by Claude using current market data. Click "↻ Regenerate" any time for a fresh read. Use it to inform your weekly strategy.
